The Problem

I love classical music!

Easy to find songs that are popular.

Finding similar songs can be hard.

Spotify's genre data is woefully inadequate.

Does not really perform well...


What's the concept behind my playlists?


Rich metadata from Spotify

Multiple recordings for one song
Here's what I did
  • Collected metadata of ~80,000 songs from Spotify Web API
  • Organized metadata in PostgreSQL
  • Fuzzy matching, clause matching for songs
  • Example: Fur Elise; Fur Elise, WoO 59; Bagatelle in A Minor, WoO 59 - "Fur Elise":Poco moto
  • Take song matches and treat as nodes
  • Treat sharing an album as a weighted link between nodes
  • ~45,000 Nodes and ~550,000 Edges implemented in igraph
The Algorithm

Perform unsupervised community detection clustering with Infomap

No labels attached to communities

Majority voting on communities with labeled data
